# 12A4 Website

## Overview
The 12A4 website is a project dedicated to preserving and showcasing the memories of the 12A4 class from K58-MDA. This website serves as a digital yearbook, featuring profiles of class members, achievements, and memorable events throughout the school years.

## Project Structure
The project is organized into several directories and files:

- **src/**: Contains the source code for the website.
  - **index.html**: The main HTML file that serves as the entry point.
  - **styles/**: Contains CSS files for styling the website.
    - **base.css**: Base styles including typography and colors.
    - **layout.css**: Layout styles using grid and flexbox.
    - **components.css**: Styles for UI components like buttons and cards.
  - **scripts/**: Contains JavaScript files for functionality.
    - **main.js**: Initializes website functionality and handles events.
    - **ui.js**: Functions for user interface interactions.
    - **analytics.js**: Tracks user interactions for analytics.
  - **components/**: Contains reusable HTML components.
    - **header.html**: Structure for the website's header.
    - **nav.html**: Structure for the navigation menu.
    - **footer.html**: Structure for the footer.
  - **data/**: Contains data files.
    - **members.json**: Data related to class members.

- **public/**: Contains public assets.
  - **fonts/**: Font files used in the website.
  - **icons/**: Icon files used in the UI.

- **package.json**: Configuration file for npm, listing dependencies and scripts.

- **.gitignore**: Specifies files and directories to be ignored by Git.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd 12a4-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in a web browser to view the website.

## Features
- **Dynamic Member Profiles**: View profiles of class members with achievements and contributions.
- **Event Timeline**: A timeline showcasing important events throughout the school years.
- **Interactive Games**: Play games and engage with classmates through interactive features.
- **Video Gallery**: Watch videos capturing memorable moments of the class.
